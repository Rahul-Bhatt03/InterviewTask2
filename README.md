Dashboard Web Application

This is a demo frontend project created as part of a technical screening round. The project demonstrates a blog-style dashboard where an authenticated user can create, update, delete, and view blog posts. Reading post details by ID is not implemented since it was not included in the task requirements.

## Setup and Run

1. Clone the repository:
git clone <repo-url>

2. Navigate into the project directory:
cd frontend

3. Install dependencies:
npm install

4. Run the development server:
npm run dev
